# Alfred

**Your personal computer assistant, powered by Claude Cowork.**

Alfred is a deployable, portable AI assistant configuration — 27 skills, 8 service connectors, and a bootstrap system that scans any Windows machine and adapts to it. Clone this repo onto a new computer, run the setup, and Alfred is ready to work.

Built by Chris Cimino. Originally deployed on the AceMagic Warlord Mini W1.

## Quick Start

New machine? See **[SETUP.md](SETUP.md)** — a step-by-step guide that assumes no coding knowledge.

Already set up and just want to update? `git pull` and re-run the install script below.
